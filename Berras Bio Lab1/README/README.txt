Berra Bio Lab 1 Inlämning READEME:

I appsettings.json ligger connectionstringen för db. 

Jag kör inte local db utan sqlexpress. Men programmet är skriven med "code first".

1. Byt ut connectionstringen.
2. Kör en "update-database" i package manager console.
3. Kör script "Add data to database".
4. Testa på!

I mappen som denna README fil ligger i så finns också ett script för att lägga till data i servern.
